How-to blog with snippets
=========================

Install:
--------

###Windows:

1. Install Anaconda 32 bit with Python 3.5.

2. Use pip to install requirements:

    pip install pelican
    pip install -r requirements.txt

3. Test:

    fab reserve
    start http://localhost:8000

4. Publish:

    fab republish
